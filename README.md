# 🎓 Student Result Management System

## Overview
A complete Student Result Management System that combines SQL database, Python logic, and Excel reporting.

## Features
- ✅ SQLite database for storing students, subjects, and results
- ✅ Automatic grade calculation (A+, A, B+, B, C, D, F)
- ✅ Class rankings and performance analysis
- ✅ Excel report generation with xlwings
- ✅ Interactive command-line interface

## Tech Stack
- **Python** - Core logic
- **SQLite** - Database management
- **xlwings** - Excel integration
- **pandas** - Data manipulation
